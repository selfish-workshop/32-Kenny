# InterLink — 핵심 기능 상세 명세

> MVP 범위 정의 기반으로 작성된 기능 명세서입니다.

---

## F-01. 회원가입 & 로그인

### 개요
기업 담당자와 통역사가 각자의 역할로 가입하고 인증하는 기능

### 세부 기능

#### 가입 방식
| 방식 | 제공 여부 |
|------|-----------|
| 이메일 + 비밀번호 | ✅ |
| 구글 소셜 로그인 | ✅ |
| 카카오 소셜 로그인 | ✅ |

#### 역할 선택
- 가입 첫 단계에서 **기업 담당자 / 통역사** 중 선택
- 역할에 따라 이후 입력 폼 분기

#### 기업 담당자 가입 필드
| 필드 | 필수 여부 | 비고 |
|------|-----------|------|
| 이름 | ✅ | |
| 이메일 | ✅ | 인증 메일 발송 |
| 비밀번호 | ✅ | 8자 이상, 영문+숫자 조합 |
| 회사명 | ✅ | |
| 사업자등록번호 | ✅ | 공공API 실시간 검증 |
| 연락처 | ✅ | |

#### 통역사 가입 필드
| 필드 | 필수 여부 | 비고 |
|------|-----------|------|
| 이름 | ✅ | |
| 이메일 | ✅ | 인증 메일 발송 |
| 비밀번호 | ✅ | 8자 이상, 영문+숫자 조합 |
| 연락처 | ✅ | |

> 통역사 상세 정보(언어, 지역, 전문분야 등)는 가입 완료 후 프로필 작성에서 입력

### 비즈니스 룰
- 이메일 인증 완료 전까지 로그인 불가
- 사업자등록번호 검증 실패 시 가입 불가 (오류 메시지 표시)
- 소셜 로그인 시에도 역할 선택 + 추가 정보(회사명, 사업자번호 등) 입력 필요

## F-02. 통역사 등록 & 자격 심사

### 개요
통역사가 이력서와 어학 자격증을 제출하면 관리자가 심사 후 승인/반려하는 기능

### 세부 기능

#### 서류 제출
| 서류 | 필수 여부 | 형식 |
|------|-----------|------|
| 이력서 | ✅ | PDF, DOC, DOCX (최대 10MB) |
| 어학 자격증 사본 | ✅ | PDF, JPG, PNG (최대 10MB) |

#### 심사 프로세스
```
통역사 서류 제출
    ↓
관리자 검토 (어드민 페이지)
    ↓
승인 → 프로필 공개 상태로 전환 + 이메일 알림
반려 → 반려 사유 기재 + 이메일 알림 → 수정 후 재신청 가능
```

#### 심사 상태
| 상태 | 설명 |
|------|------|
| 심사 대기 | 서류 제출 완료, 관리자 검토 전 |
| 심사 중 | 관리자 검토 중 |
| 승인 | 프로필 공개, 예약 수락 가능 |
| 반려 | 재신청 가능, 반려 사유 제공 |

### 비즈니스 룰
- 심사 미통과 상태에서는 프로필이 검색에 노출되지 않음
- 반려 후 서류 수정하여 재신청 가능 (횟수 제한 없음)
- 관리자는 반려 사유를 반드시 입력해야 처리 완료

## F-03. 통역사 프로필

### 개요
승인된 통역사의 공개 프로필 — 기업 담당자가 검색 및 예약 시 조회하는 페이지

### 프로필 필드
| 필드 | 필수 여부 | 설명 |
|------|-----------|------|
| 프로필 사진 | 선택 | |
| 이름 | ✅ | |
| 한 줄 소개 | ✅ | 최대 100자 |
| 상세 소개 | 선택 | 최대 500자 |
| 통역 가능 언어 | ✅ | 복수 선택 |
| 활동 가능 지역 | ✅ | 대륙 > 국가 > 도시 (복수 선택) |
| 전문 분야 | ✅ | 산업군 복수 선택 (전자, 뷰티, 식품, 기계, 화학 등) |
| 경력 연차 | ✅ | 숫자 입력 |
| 성별 | ✅ | 남 / 여 |
| 권장 단가 | 자동 표시 | 플랫폼 언어·지역별 가이드라인 기준 |
| 후기 수 / 평균 별점 | 자동 집계 | 리뷰 데이터 기반 |

### 비즈니스 룰
- 관리자 승인 완료 후에만 프로필 공개
- 통역사 본인이 언제든 프로필 수정 가능 (재심사 불필요)
- 플랫폼 권장 단가는 통역사가 수정 불가 — 관리자만 조정

## F-04. 통역사 검색 & 필터

### 개요
기업 담당자가 조건에 맞는 통역사를 탐색하는 기능

### 검색 필터
| 필터 | 유형 | 설명 |
|------|------|------|
| 지역 | 계층 선택 | 대륙 > 국가 > 도시 순 선택 |
| 언어 | 복수 선택 | 통역 가능 언어 |
| 전문 분야 | 복수 선택 | 산업군 |
| 경력 연차 | 범위 선택 | 예: 1년 미만 / 1~3년 / 3~5년 / 5년 이상 |
| 성별 | 단일 선택 | 전체 / 남 / 여 |

### 정렬 기준
- 후기 많은 순
- 평점 높은 순
- 최근 등록 순

### 비즈니스 룰
- 승인된 통역사만 검색 결과에 노출
- 필터 미선택 시 전체 통역사 목록 표시
- 검색 결과에서 프로필 카드 클릭 시 상세 페이지 이동

## F-05. 예약 시스템

### 개요
기업 담당자가 통역사에게 예약을 요청하고, 통역사가 수락/거절하며 확인서를 발행하는 기능

### 예약 요청 입력 정보
| 필드 | 필수 여부 | 설명 |
|------|-----------|------|
| 전시회명 | ✅ | 텍스트 입력 |
| 전시 기간 | ✅ | 날짜 범위 선택 |
| 필요 시간 | ✅ | 시작 시간 ~ 종료 시간 |
| 요청 내용 | ✅ | 상세 요청 텍스트 (최대 500자) |
| 예상 단가 | 선택 | 협의용 참고값 |

### 예약 상태 흐름
```
요청 대기
    ↓ (통역사 응답 or 48시간 경과)
수락됨 → 확인서 자동 발행 → 이메일 발송
거절됨 → 기업에 이메일 알림
자동 거절 (48시간 무응답) → 기업에 이메일 알림
    ↓ (예약 진행 중)
완료 → 리뷰 작성 요청 이메일 발송
취소 (기업 or 통역사) → 상대방 이메일 알림
```

### 예약 상태 정의
| 상태 | 설명 |
|------|------|
| 요청 대기 | 기업이 요청 발송, 통역사 미응답 |
| 수락됨 | 통역사 수락, 확인서 발행 완료 |
| 거절됨 | 통역사 거절 또는 48시간 자동 거절 |
| 완료 | 전시 종료 후 완료 처리 |
| 취소 | 기업 또는 통역사가 취소 |

### 타임아웃 정책
- 통역사가 **48시간 내 미응답** 시 자동 거절 처리
- 자동 거절 시 기업에 이메일 알림 발송

### 취소 정책
- 기업·통역사 모두 취소 가능
- 별도 패널티 없음 (MVP 단계)
- 취소 시 상대방에게 이메일 알림

### 확인서(PDF) 포함 정보
| 항목 |
|------|
| 예약 번호 |
| 기업명 / 담당자명 |
| 통역사명 |
| 전시회명 / 전시 기간 / 필요 시간 |
| 통역 언어 |
| 권장 단가 |
| 발급일 |

### 비즈니스 룰
- 동일 통역사에게 동일 시간대 중복 예약 불가 (시스템 자동 방지)
- 확인서는 기업·통역사 모두 다운로드 가능
- 완료 처리는 전시 종료일 기준 자동 전환 또는 관리자 수동 처리

## F-06. 리뷰

### 개요
예약 완료 후 기업이 통역사에게 후기를 남기는 기능

### 리뷰 항목
| 항목 | 설명 |
|------|------|
| 종합 별점 | 5점 만점 |
| 텍스트 후기 | 선택 입력, 최대 300자 |

### 비즈니스 룰
- 예약 상태가 **완료**인 경우에만 리뷰 작성 가능
- 예약 1건당 리뷰 1회만 작성 가능
- 작성된 리뷰는 통역사 프로필에 공개
- 완료 후 리뷰 작성 요청 이메일 자동 발송 (링크 포함)

## F-07. 이메일 알림

### 발송 시점 및 수신자

| 이벤트 | 발신 | 수신 |
|--------|------|------|
| 회원가입 이메일 인증 | 시스템 | 가입 사용자 |
| 통역사 심사 승인 | 시스템 | 통역사 |
| 통역사 심사 반려 | 시스템 | 통역사 (반려 사유 포함) |
| 예약 요청 도착 | 시스템 | 통역사 |
| 예약 수락 | 시스템 | 기업 담당자 (확인서 PDF 첨부) |
| 예약 거절 | 시스템 | 기업 담당자 |
| 예약 자동 거절 (48시간) | 시스템 | 기업 담당자 |
| 예약 취소 | 시스템 | 상대방 (기업 or 통역사) |
| 예약 완료 + 리뷰 요청 | 시스템 | 기업 담당자 (리뷰 작성 링크 포함) |

## F-08. 관리자 페이지

### 개요
플랫폼 운영자가 서비스를 관리하는 내부 도구

### 기능 목록

#### 통역사 이력서 심사
- 대기 중인 심사 목록 조회
- 제출 서류(이력서, 자격증) 미리보기
- 승인 / 반려 처리 (반려 시 사유 입력 필수)

#### 회원 관리
- 기업·통역사 전체 회원 목록 조회
- 회원 상세 정보 조회
- 계정 정지 / 정지 해제

#### 예약 현황 조회
- 전체 예약 리스트 조회
- 상태별 필터 (요청 대기 / 수락 / 거절 / 완료 / 취소)
- 예약 상세 정보 조회

#### 분쟁 처리
- 신고 접수 목록 조회
- 신고 내용 확인 및 처리 상태 관리 (접수 / 처리 중 / 완료)
- 처리 결과 메모 작성

#### 공지사항 관리
- 공지사항 작성 / 수정 / 삭제
- 공개 여부 설정

#### 권장 단가 관리
- 언어·지역별 권장 단가 조회 및 수정

## 기능 간 의존 관계

```
F-01 회원가입
    └→ F-02 통역사 등록·심사 (통역사 역할)
           └→ F-03 통역사 프로필 공개
                  └→ F-04 검색·필터
                         └→ F-05 예약 시스템
                                └→ F-06 리뷰

F-07 이메일 알림 ← F-02, F-05, F-06 이벤트 구독
F-08 관리자 페이지 ← F-02, F-01, F-05 데이터 관리
```
