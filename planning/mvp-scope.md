# InterLink — MVP 범위 정의

> 브레인스토밍 기반으로 확정된 MVP 범위입니다.

---

## 1. MVP 핵심 원칙

- 결제는 제외 — 계좌이체 등 플랫폼 외부에서 처리
- 한국어 단일 언어 서비스
- 통역사 자격은 관리자가 수동으로 이력서 검토 후 승인

---

## 2. 사용자 유형

| 유형 | 설명 |
|------|------|
| **기업 담당자** | 해외 trade show 참가 중소기업의 통역 섭외 담당자 |
| **통역사** | 전시 개최지 인근의 유학생·교민 |
| **관리자** | 플랫폼 운영자 (통역사 심사, 분쟁 처리 등) |

---

## 3. 타겟 지역

전 세계 전시회 개최지 기준 (easyfair.co.kr 등록 대륙·국가·도시)

| 대륙 | 국가 | 주요 도시 |
|------|------|-----------|
| 유럽 | 독일 | 프랑크푸르트, 쾰른, 뒤셀도르프, 하노버, 뮌헨, 베를린 |
| 유럽 | 프랑스 | 파리 |
| 유럽 | 이탈리아 | 밀라노 |
| 유럽 | 영국 | 런던 |
| 유럽 | 스페인 | 바르셀로나 |
| 북미 | 미국 | 라스베가스, 시카고, 뉴욕 |
| 아시아 | 중국 | 상하이, 광저우 |
| 아시아 | 일본 | 도쿄 |
| 아시아 | 싱가포르 | 싱가포르 |
| 중동 | UAE | 두바이 |

→ 통역사 프로필에 **활동 가능 대륙·국가·도시** 필드로 관리

---

## 4. MVP 포함 기능

### 4-1. 회원가입 & 로그인
- 이메일 + 비밀번호
- 소셜 로그인 (구글, 카카오)
- 역할 구분: 기업 / 통역사 선택 후 가입

### 4-2. 통역사 등록 & 심사
- 이력서(파일) 업로드
- 관리자 수동 승인/반려
- 승인 완료 시 프로필 공개

### 4-3. 통역사 프로필
| 필드 | 내용 |
|------|------|
| 기본 정보 | 이름, 사진, 소개 |
| 언어 | 통역 가능 언어 (복수 선택) |
| 활동 지역 | 대륙 > 국가 > 도시 |
| 전문 분야 | 산업군 (전자, 뷰티, 식품 등) |
| 경력 연차 | 연차 입력 |
| 성별 | 선택 |
| 권장 단가 | 플랫폼 제공 언어·지역별 가이드라인 기준 |

### 4-4. 통역사 검색 & 필터
| 필터 | 옵션 |
|------|------|
| 지역 | 대륙 > 국가 > 도시 |
| 언어 | 통역 언어 |
| 전문 분야 | 산업군 |
| 경력 연차 | 범위 선택 |
| 성별 | 전체 / 남 / 여 |
| 후기 수 | 정렬 기준으로 활용 |

### 4-5. 예약 시스템
1. **연결 요청**: 기업이 통역사에게 예약 요청 발송
2. **수락/거절**: 통역사가 요청 확인 후 응답
3. **일정 관리**: 전시회 날짜·시간 기반 예약 슬롯 관리, 중복 예약 자동 방지
4. **확인서 발급**: 예약 확정 시 플랫폼이 확인서(PDF) 자동 생성·발행

### 4-6. 리뷰
- 예약 완료 후 기업 → 통역사 단방향 리뷰
- 별점 + 텍스트

### 4-7. 알림
- 이메일 알림 (예약 요청 / 수락 / 거절 / 완료 이벤트)

### 4-8. 관리자 페이지
| 기능 | 설명 |
|------|------|
| 통역사 이력서 심사 | 승인 / 반려 처리 |
| 회원 관리 | 기업·통역사 계정 조회·정지 |
| 예약 현황 조회 | 전체 예약 리스트 및 상태 확인 |
| 분쟁 처리 | 신고 접수 및 처리 |
| 공지사항 관리 | 공지 작성·수정·삭제 |

---

## 5. MVP 제외 기능 (향후 고려)

| 기능 | 이유 |
|------|------|
| 결제 시스템 | 초기 운영 복잡도 절감, 계좌이체로 대체 |
| 통역사 → 기업 리뷰 | 단방향으로 시작, 데이터 쌓인 후 추가 |
| 인앱 알림 | 이메일로 충분한 MVP 단계 |
| 카카오 알림톡 | 이메일 안정화 후 추가 |
| 다국어 지원 | 한국어 단일로 시작 |
| 자동 자격 심사 | 수동 운영으로 신뢰도 확보 우선 |

---

## 6. 플랫폼 단가 가이드라인 (기준안)

> 통역사가 단가를 직접 설정하지 않고, 플랫폼이 언어·지역별 권장 단가를 제공

| 언어권 | 지역 | 권장 시간당 단가 (참고) |
|--------|------|------------------------|
| 영어 | 북미·유럽 | 협의 예정 |
| 독일어 | 독일 | 협의 예정 |
| 중국어 | 아시아 | 협의 예정 |
| 일본어 | 일본 | 협의 예정 |
| 기타 | - | 협의 예정 |

→ 구체적인 단가 기준은 전략팀과 협의 후 확정
