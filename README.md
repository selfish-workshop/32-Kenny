# InterLink

> 해외 trade show 참가 중소기업과 현지 유학생·교민 통역을 연결하는 매칭 플랫폼

## 서비스 개요

중소기업이 해외에서 개최되는 trade show에 참가할 때, 개최지 근처의 유학생·교민을 통역으로 채용할 수 있도록 매칭해주는 웹 플랫폼입니다.

- **기업**: 현지 통역 인력을 빠르고 신뢰할 수 있는 방식으로 확보
- **통역사**: 전문성을 살려 부업 기회를 얻고, 커리어를 쌓는 플랫폼

## 핵심 기능

| # | 기능 | 설명 |
|---|------|------|
| 1 | 통역사 자격 심사 | 후보자가 이력서를 업로드하면 관리자가 자격을 검토·승인 |
| 2 | 프로필 & 검색 | 언어, 위치, 전문 분야별 통역사 검색 및 프로필 조회 |
| 3 | 매칭 & 예약 | trade show 일정에 맞춰 통역사 매칭 및 예약 |
| 4 | 리뷰 / 평점 | 이용 후 통역사 평가 및 후기 작성 |
| 5 | 결제 시스템 | 플랫폼 내 안전한 보수 지급 |

## 폴더 구조

```
workshop-github/
├── README.md         — 프로젝트 소개 (현재 파일)
├── PRD.md            — 제품 요구사항 문서
├── planning/         — 무엇을 만들 것인가 (기획팀)
├── design/           — 어떻게 보여줄 것인가 (디자인팀)
├── cx/               — 누가 어떻게 사용하는가 (CX팀)
├── marketing/        — 어떻게 알릴 것인가 (마케팅팀)
└── strategy/         — 어떻게 지속할 것인가 (전략팀)
```
